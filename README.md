## ALGORITHM
### STEP-1
  Open notepad and type the HTML code.

### STEP-2
  Apply CSS border style and color for the heading.

### STEP-3
  Set the alignments, font colors and styles for the paragraph tag.

### STEP-4
  Open the file in a browser and verify the output.
